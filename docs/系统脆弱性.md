# 系统脆弱性 (System Fragility)

## 定义
故意设计脆弱、易受干扰的系统，观察其崩溃与重生。系统脆弱性是混沌媒体艺术中重要的设计策略，通过引入不稳定性来探索系统的边界和潜力。

## 理论背景
- 受到纳西姆·塔勒布的"反脆弱"理论启发
- 与复杂系统理论中的临界性概念相关
- 体现了对稳定性的重新思考

## 在混沌媒体艺术中的应用
- 作为系统设计的指导原则
- 帮助发现系统的意外行为
- 探索系统崩溃的美学价值

## 设计策略
- 减少系统的容错能力
- 引入敏感的参数设置
- 创建相互依赖的脆弱组件

## 相关概念
- 反脆弱性
- 临界性
- 系统崩溃美学

## 哲学意义
系统脆弱性挑战了传统设计中对稳定性和可靠性的追求，转而探索不稳定状态下的创造性潜力，体现了对变化和转化的重视。
