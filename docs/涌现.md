# 涌现 (Emergence)

## 定义
系统层面出现的整体性质，无法从组成部分的性质中预测。涌现现象是复杂系统的重要特征，指的是整体大于部分之和的现象。

## 理论背景
- 涌现分为弱涌现和强涌现两种类型
- 弱涌现可以通过系统仿真来预测
- 强涌现具有根本的不可预测性

## 在混沌媒体艺术中的应用
- 作为评价作品成功与否的重要标准
- 指导系统设计以促进有价值模式的出现
- 帮助理解创作过程中意外成果的产生

## 相关概念
- 自组织
- 复杂性
- 集体行为
