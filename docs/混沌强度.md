# 混沌强度 (Chaos Intensity)

## 定义
衡量系统偏离预定路径程度的参数化指标。混沌强度是一个可调节的参数，用于控制创作过程中不可预测性的程度。

## 理论背景
- 基于混沌理论中的参数控制概念
- 类似于动力系统中的分岔参数
- 与系统复杂度和扰动频率相关

## 参数范围
- 低强度 (1-3): 轻微偏离预期结果
- 中强度 (4-6): 明显但可控的偏差
- 高强度 (7-10): 极度不可预测的结果

## 在混沌媒体艺术中的应用
- 作为创作过程中的调节工具
- 帮助创作者控制最终作品的可预测性
- 平衡创作意图与意外发现

## 相关概念
- 系统复杂度
- 扰动频率
- 控制参数
