# 混沌配方 (Chaos Recipes)

## 定义
一套可操作的方法论，而非随机行为。混沌配方是混沌媒体艺术中的具体创作指导，提供结构化的混乱引入方式。

## 理论背景
- 受到烹饪艺术中配方概念的启发
- 体现了结构化与随机性的结合
- 与算法艺术中的参数化方法相关

## 核心公式
```
混沌强度 = 系统复杂度 × 失控容忍度 × 干预时机

高价值混乱 = 有意义的随机 + 可感知的模式 + 情感冲击
```

## 在混沌媒体艺术中的应用
- 作为创作过程中的参考框架
- 帮助创作者调节混沌的程度
- 提供可重复的实验基础

## 实践策略
- 参数调节：调整混沌强度参数
- 时机控制：选择合适的干预时机
- 效果评估：评估混乱产生的价值

## 相关概念
- 参数化创作
- 随机算法
- 结构化实验

## 哲学意义
混沌配方体现了在看似矛盾的"结构化混乱"中寻找平衡的理念，证明了即使在追求不确定性时也需要一定的指导原则。
