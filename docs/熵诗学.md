# 熵诗学 (Entropy Poetics)

## 定义
将热力学第二定律和信息论概念应用于美学创作的理论框架。熵诗学探讨了在创作过程中无序性与有序性之间的动态关系，以及这种关系如何影响审美体验。

## 理论背景
- 熵的概念最初来自热力学，表示系统的无序程度
- 在信息论中，熵衡量信息的不确定性
- 熵诗学将这些科学概念转化为创作策略

## 在混沌媒体艺术中的应用
- 作为整个框架的理论基础
- 指导创作者在有序与无序之间找到平衡
- 为评价混沌作品提供理论依据

## 相关概念
- 信息熵
- 美学熵
- 创作中的不可逆性
