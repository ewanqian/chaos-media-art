# 混沌生态学家 (Chaos Ecologist)

## 定义
将自身角色从创作者转变为系统培育者的艺术家。混沌生态学家不再试图完全控制创作结果，而是专注于创建能够产生有趣现象的环境。

## 理论背景
- 受生态学中生态系统概念启发
- 与控制论中的观察者概念相关
- 体现了后人类主义的视角

## 角色特征
- 参数设定者：设定初始条件和边界规则
- 生态培育者：创建能让意外发生的环境
- 实时策展人：在混乱中即时筛选、组合
- 跨媒介翻译官：在不同感官语言间建立联系

## 在混沌媒体艺术中的应用
- 作为创作者的新身份定位
- 指导创作过程中的态度转变
- 重新定义艺术创作中的人机关系

## 相关概念
- 系统思维
- 生态设计
- 协作创作
