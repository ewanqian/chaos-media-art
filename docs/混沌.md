# 混沌 (Chaos)

## 定义
在数学和物理语境中的确定性非线性动力学系统，表现出对初始条件的敏感依赖性，而非简单的"混乱"或"无序"。混沌系统具有三个关键特征：对初始条件的敏感依赖性、拓扑传递性和周期轨道的稠密性。

## 理论背景
- 混沌理论起源于Edward Lorenz在气象学中的研究
- 蝴蝶效应是混沌系统中最著名的概念之一
- 混沌不同于纯粹的随机性，它是确定性系统中产生的看似随机的行为

## 在混沌媒体艺术中的应用
- 作为创作方法论的基础
- 指导系统设计中的参数设置
- 帮助理解创作过程中涌现现象的产生机制

## 参考文献
